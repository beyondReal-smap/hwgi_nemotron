# 에이전트 가이드

> AI 에이전트가 세션을 시작할 때 읽는 문서입니다.

---

## 작업 원칙

- 모든 커뮤니케이션은 **한국어**로
- **최소 변경**: 꼭 필요한 범위만 수정
- **근본 원인 해결** 우선, 우회 패치 지양
- 기존 코드 스타일 준수
- 커밋 메시지: `feat/fix/docs/refactor/chore` 형태
- **데이터셋 라이선스 준수**: `nvidia/Nemotron-Personas-Korea`의 라이선스/이용 약관을 항상 확인하고 인용
- **개인정보 취급**: 페르소나는 합성 데이터지만, 실제 상품·약관과 결합되는 만큼 PII가 포함된 로그/캐시는 남기지 않기

---

## 용어 정리

| 용어 | 설명 |
|------|------|
| **페르소나(Persona)** | Nemotron-Personas-Korea의 합성 한국인 인물 프로필. 인구통계·라이프스타일·지역 등 속성 보유 |
| **반응도(Affinity Score)** | 특정 상품에 대해 특정 페르소나가 반응할 가능성을 0–1 또는 0–100으로 정량화한 점수 |
| **소구점(Selling Point)** | 상품설명서·약관에서 LLM이 추출한 핵심 가치/혜택 키워드 |
| **공략 지역(Target Region)** | 반응도 높은 페르소나가 밀집한 행정구역(시/도/시군구) |
| **FP** | Financial Planner. 보험·금융 상품 판매를 담당하는 설계사. 본 도구의 주요 사용자 |
| **상품기획자** | 상품 설계 및 타겟 전략 수립자. 본 도구의 또 다른 주요 사용자 |
| **MCP** | Model Context Protocol. AI가 외부 도구와 통신하는 방식 |
| **P0/P1/P2** | 우선순위. P0(긴급) > P1(중요) > P2(보통) |
