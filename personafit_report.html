01. 프로젝트 개요 (Overview)

PersonaFit 엔진의 탄생 배경과 해결하고자 하는 비즈니스 가치

금융 및 보험 산업군에서 새로운 상품이 출시될 때, 상품 기획자 및 영업 설계사(FP)는 늘 동일한 숙제에 직면합니다. "이 상품은 정확히 어떤 가구를 구성하는 몇 세의 한국인들에게 가장 큰 효용을 주는가? 그리고 그들을 어떻게 공략해야 하는가?"

기존의 마케팅 방식은 기획자의 단순한 도메인 지식이나 막연한 가설에 의존하여 설계된 최소 단위의 연령대/성별 필터링에 그쳐 정교함이 결여되어 있었습니다. PersonaFit은 이러한 문제를 극복하기 위해 설계된 데이터 테크놀로지 분석 도구입니다. 복잡한 상품 설명서나 약관 텍스트를 AI로 기계 해석하고, NVIDIA가 구축한 100만 명의 고정밀 한국인 합성 페르소나 데이터셋과의 코사인 유사도 다차원 매칭을 거쳐 핵심 타겟층, 정량적 시도/시군구 분포, 영업 화법이 담긴 기획 인사이트를 20초 이내에 원스톱으로 산출해냅니다.

🎯 핵심 비즈니스 가치 제안 (Value Proposition)

복잡한 보험 약관이나 대고객 금융 상품설명서 파일을 드래그앤드롭하는 것만으로, 100만 명의 가상 한국인 데이터셋에서 가장 반응도가 강할 Top 50 페르소나를 정밀 매칭합니다. 이를 통해 신상품 기획의 실패 확률을 줄이고 리테일 FP들의 세일즈 성공률을 극대화합니다.

02. Nemotron-Personas-Korea 데이터셋 상세

NVIDIA NeMo Data Designer 기반의 100만 합성 데이터셋 분석

PersonaFit 핵심 통계 엔진의 기반이 되는 데이터셋은 nvidia/Nemotron-Personas-Korea입니다. 이 데이터셋은 한국의 인구통계학적 특성과 삶의 맥락을 완벽히 투영하기 위해 생성된 대규모 고정밀 합성 데이터(Synthetic Data)입니다.

데이터 특징 항목 상세 스펙 및 기술 사양 비즈니스적 의의
데이터 규모 총 1,000,000행 (100만 명의 페르소나 프로필), 1.7B 토큰 한국의 거대한 모수 인구를 통계적 대표성 있게 커버
인구통계 컬럼 연령(Age), 성별(Sex), 지역(Province, District), 직업(Occupation), 학력(Education), 가구유형(Family Type) 6가지 핵심 인구통계 변수를 활용한 5차원 다면적 타겟 매칭
다차원 페르소나 텍스트 종합(Persona), 직업(Professional), 스포츠, 예술, 여행, 요리, 가족 7개 영역의 구체적 서술 텍스트 제공 단순 통계수치 이면의 생활 습관, 선호 키워드를 활용한 시멘틱 매칭
라이선스 Creative Commons Attribution 4.0 International (CC BY 4.0) 조건(출처 명시) 충족 시 영리적 목적의 제품 개발 및 서비스 적용 가능
통계적 정밀도 NVIDIA NeMo Data Designer + KOSIS / 대법원 / 국민건강보험 / KREI 통계 지표 활용 한국 실제 통계 가중치를 그대로 반영하여 가상의 왜곡 극소화

NVIDIA의 합성 데이터는 단순 무작위 생성이 아닌 한국인의 성별 비율, 각 시도 및 시군구 단위의 인원 비율, 연령별 직업 통계 매트릭스에 입각하여 Gemma 모델을 통해 파이프라인 정제되었습니다. PersonaFit은 이 강력한 데이터 셋의 7가지 텍스트 차원을 고유 임베딩 모델로 처리하여 시멘틱 유사도 대조용 인메모리 스토어로 내재화하였습니다.

03. 아키텍처 및 기술 스택 (Architecture)

현대적인 프리미엄 디자인 구현을 위한 유기적 설계 아키텍처

Next.js 14 (App Router) TypeScript Tailwind CSS FastAPI (Python 3.12) UV Package Manager NumPy PyArrow & Parquet Claude 3.5 Sonnet Claude 3.5 Haiku OpenAI Text Embedding 3 Recharts

PersonaFit은 대규모 연산(100만 행 데이터 처리)과 고도화된 LLM 파이프라인을 부드러운 사용자 경험으로 풀어내기 위해 백엔드와 프론트엔드가 명확한 단방향 의존성으로 결합된 이중 레이어 아키텍처를 가집니다.

🎨

Premium Frontend (Next.js 14)

Recharts를 탑재하여 한국의 17개 시도별 인구통계분포와 8대 세그먼트를 반응형으로 렌더링합니다. 한화(Hanwha) 디자인 시스템 톤앤매너인 Vellum(크림 배경)과 Terra(오렌지) 액센트를 조합하여 고도의 정교함을 선사합니다.

⚡

High-Performance Backend (FastAPI)

uv 패키지 매니저 기반의 고성능 API 레이어입니다. 10만 행의 고가치 Stratified 샘플링 데이터셋과 임베딩 벡터(`.npy`)를 서버 시동 시 인메모리에 로드하여 코사인 유사도(Cosine Similarity) 연산을 단 0.4초 만에 끝마칩니다.

🧠

Differentiated LLM Orchestration

Claude 3.5 Sonnet은 Tool Use 강제를 통해 정교한 5차원 타겟 JSON을 추출합니다. Claude 3.5 Haiku는 타겟 페르소나 정보를 넘겨받아 FP용 정성적 세일즈 화법과 리포트를 극도로 빠르게 작성합니다.

04. 타겟 매칭 & 분석 프로세스

설명서 업로드부터 영업 제안서 생성까지의 5단계 정밀 파이프라인

01

상품 분석 및 소구점 추출 (Claude 3.5 Sonnet)

업로드된 상품설명서/약관 파일(PDF, HWPX, HWP 등)에서 텍스트를 자동 파싱한 뒤, Sonnet 모델이 5대 타겟 디렉티브(연령대, 성별, 가구 유형, 학력 수준, 선호 직업군)와 페르소나 가중치 벡터를 정합성 있는 스키마로 정확하게 추출해냅니다.

02

고정밀 다차원 룰 필터링 (Rule-based Constraint)

1단계에서 생성된 타겟 조건을 기반으로 데이터베이스 내의 100k 페르소나 풀에 1차 필터링을 가합니다. 특정 조건이 지나치게 좁아 매칭 풀이 200명 이하로 고갈되는 경우, 엔진이 실시간으로 룰 제약 요건을 안전하게 스케일 백(완화)하여 편향을 미연에 방지합니다.

03

시멘틱 벡터 임베딩 매칭 (OpenAI Embedding & NumPy)

상품 소구점 텍스트를 `text-embedding-3-small` (1536차원) 모델을 통해 실시간 임베딩 처리합니다. 이후 NumPy의 행렬 연산 라이브러리를 동원해 1차 룰 필터를 거쳐 생존한 타겟 후보군과의 코사인 유사도(Cosine Similarity)를 대규모 고속 연산하여 최상위 200명을 발굴합니다.

04

하이브리드 스코어링 결합 (Hybrid Fusion Scoring)

단순 텍스트 매칭 오류를 메우기 위해 독자적인 복합 스코어 수식을 사용합니다:
최종 스코어 = 0.7 × (코사인 유사도) + 0.2 × (인구통계 룰 보너스) + 0.1 × (페르소나 카테고리 보너스)
이를 통해 군인, 특정 자영업 등 키워드만으로 잡히지 않는 통계적 부합성을 가진 진짜 타겟 페르소나를 Top 50으로 정밀 정렬합니다.

05

대시보드 시각화 및 인사이트 생성 (Claude 3.5 Haiku)

상위 50인의 정량 데이터를 Recharts를 통해 동적 대시보드로 시각화하는 동시에, Claude Haiku 모델이 이들의 심리적 장벽, 구매 결정 요인, 설득 프로세스 및 구체적인 롤플레잉용 영업 화법 스크립트를 포함한 미려한 마크다운 리포트를 즉시 생성하여 제공합니다.

05. 비즈니스 현실에서의 활용 방안 (Practical Use Cases)

페르소나 핏 엔진을 즉시 투입 가능한 금융 비즈니스 시나리오

📋

금융/보험 상품 기획 피드백 루프

신상품 약관 시안을 정식 출시 전에 엔진에 투입하여 통계적으로 어떠한 페르소나 계층에서 반응도(유사도)가 튀는지 사전 검증합니다. 초기 설정했던 타겟 집단과 머신러닝 매칭 결과가 다를 경우, 소구 문구와 담보 구성을 전략적으로 재조정할 수 있습니다.

🎙️

설계사 영업 영업 화법(Sales Talk) 가이드

리테일 설계사들이 아파트 단지나 특정 지자체 등의 지역 가망 고객을 만날 때, 본 엔진을 활용하여 해당 지역 및 세그먼트의 페르소나 성향에 맞춤화된 접근 멘트와 우려 극복 화법을 사전에 학습하여 청약 성공률을 비약적으로 높입니다.

📍

지방 리테일 마케팅 & 지역 타겟팅

공략 지역 분석 탭을 통해 상위 반응도를 보인 광역 시도 및 시군구 순위를 직관적으로 파악합니다. 예컨대 '경북 칠곡군', '제주 제주시' 등 통계 기반 타겟 농도가 극도로 높은 행정 구역에 리플렛 배포나 지점 판촉 자원을 집중 투입해 마케팅 투자자본수익률(ROI)을 향상시킵니다.
